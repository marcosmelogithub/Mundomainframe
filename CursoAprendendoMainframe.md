### Tópicos
---

**TSO**

Ambiente de navegação que permite ao usuario operar arquivos, acessar banco de dados, criar programas Cobol,JCL,REXX, verificar e corrigir erros de execução de programas.

---
**JCL**

Principal linguagem/script para execucão de programas Cobol,Assembler, REXX. Também utilizado para operações de arquivos sequenciais e indexados (VSAM).

---
**COBOL**

Principal linguagem no mainframe para execucão de programas que fazem tratamento de arquivos (leitura, mesclagem, gravacao).

---
**SQL/DB2**

Linguagem para manuseio de dados relacionais. O DB2 é o principal sistema de gestão de dados relacionais (tabelas, chaves , relações).

---
**DFSORT**

Atualmente o principal utilitário para manuseio de arquivos de dados (Datasets). Leitura, Mesclagem, Alteração, Exclusão , Relatórios de arquivo de dados. Pode em determinadas situações substituir programas legados de Cobol.

---
**ICETOOL**

Utilitário complementar ao DFSORT tendo como principal aplicação a geração de Relatórios personalizados a partir de arquivos sequenciais/VSAM.

---
**REXX**

Linguagem procedural destinada a tratamento de arquivos sequenciais/VSAM, pilhas internas, interaçao com ambientes MVS/USS, criação e uso de paineis para criação e PANELS de iteração com o usuario

---
**ZOWE**

Zowe é uma estrutura de software livre integrada para z/OS. Oferece interfaces modernas para interagir com o z/OS como Visual Studio,alternativa ao tradicional TSO. O Zowe é composto por vários componentes, cada um melhorando a capacidade de aprendizagem, acessibilidade e possibilidade de desenvolvimento de mainframe.
